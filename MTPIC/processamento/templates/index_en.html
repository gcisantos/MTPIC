Welcome to MTPIC!

Text Mining for Identification of Contents.




What does this system do?

The main purpose of this system is to allow the extraction of content from a text with its concepts and display them in the form of graphs.



How is this done?

Through the use of text mining techniques, data mining preprocessing text and some mathematical algorithms, it is possible to perform such operations of pattern identification and text comparison.



How is this system divided?

The system is subdivided into modules to facilitate reuse and understanding of how the entire process is performed. So we have the following modules:


● Tokenizer

This module is responsible for transforming text into a group of tokens (a word-for-word array), so later methods can more easily process that text.

● StopWords Removal

This module is responsible for cleaning words in text that have no semantic meaning, thus making the text cleaner for processing.

● Stemming

This module is responsible for performing the process of stemization of words that consists of obtaining the root form, that is, the character set responsible for giving rise to that word.

● Central Theme Extractor

This module allows the identification of the central theme of a given text, using the above modules together with a knowledge base trained with data from Wikipédia.


● Concept Extractor

This module allows the identification of the possible concepts of a certain part of a text, the same strategy of obtaining a central theme.


● Complete Miner

This is the complete system that integrates all the modules above, making it possible to identification of the central theme of a text, the identification of the concepts present in it and the display in the form of Graphs.


● View Processed Data

Through this module it is possible to search for data that has already been processed and is stored in the application database.
