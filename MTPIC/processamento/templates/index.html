Bem vindo ao MTPIC !

Mineração de Texto para Identificação de Conteúdos.




O que esse sistema faz?

O principal objetivo desse sistema é permitir que seja possivel a extração de conteúdo de um texto junto aos seus conceitos e exibir os mesmos em forma de grafos.



Como isso é feito?

Através da utilização de técnicas de mineração de texto, mineração de dados pré-processamento de texto e alguns algoritmos matemáticos é possivel realizar essas operações de identificação de padrões e comparação em texto.



De que forma esse sistema é dividido?

O sistema está subdividido em módulos para facilitar a reutilização e o entendimento de como todo o processo é realizado. Sendo assim temos os seguintes módulos:


● Tokenizer

Esse módulo é responsável por transformar o texto em um grupo de tokens(um array de palavra por palavra), dessa forma métodos posteriores podem processar mais facilmente esse texto.


● Remoção de StopWords

Esse módulo é responsável por realizar a limpeza de palavras no texto que não possuem significado semnâtico, tornando assim o texto mais limpo para o seu processamento.


● Stemização

Esse módulo é responsável por realizar o processo de stemização das palavras que consiste em obter a forma raiz, ou seja, o conjunto de caracteres responsável por dar origem aquela palavra.


● Extrator de Tema Central

Esse módulo permite realizar a identificação do tema central de um determinado texto, através da utilização dos modulos acima em conjunto com uma base de conhecimento treinada com dados do wikipédia.


● Extrator de Conceitos

Esse módulo permite realizar a identificação dos possivéis conceitos de um determinado trecho de um texto, a mesma estratégia da obtenção de tema central.


● Minerador Completo

Esse é o sistema completo que realiza a integração de todos os módulos acima, tornando possivel a identificação do tema central de um texto, a identificação dos conceitos presentes no mesmo e a exibição em forma de Grafos.


● Visualizar Dados Processados

Através desse módulo é possivel buscar dados que já foram processados e estão armazenados na base de dados da aplicação.
