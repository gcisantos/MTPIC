Bienvenido a MTPIC!

Minería de texto para Identificación de Contenidos.




¿Qué hace este sistema?

El principal objetivo de este sistema es permitir que sea posible la extracción de contenido de un texto junto a sus conceptos y mostrar los mismos en forma de grafos.



¿Cómo se hace?

Mediante la utilización de técnicas de minería de texto, minería de datos pre-procesamiento de texto y algunos algoritmos matemáticos es posible realizar estas operaciones de identificación de patrones y comparación en texto.



¿Cómo se divide ese sistema?

El sistema está subdividido en módulos para facilitar la reutilización y el entendimiento de cómo se realiza todo el proceso. Por lo tanto, tenemos los siguientes módulos:


● Tokenizer

Este módulo es responsable de transformar el texto en un grupo de tokens (una matriz de palabra por (por ejemplo, en el caso de que se produzca un error).

● Eliminación de StopWords

Este módulo es responsable de realizar la limpieza de palabras en el texto que no tienen significado y que el texto más limpio para su procesamiento.


● Stemming

Este módulo es responsable de realizar el proceso de derivación de las palabras que consiste en obtener la de forma raíz, es decir, el conjunto de caracteres responsable de dar origen a esa palabra.


● Extractor de Tema Central

Este módulo permite realizar la identificación del tema central de un determinado texto, a través el uso de los módulos arriba en conjunto con una base de conocimiento entrenada con datos de la wikipedia.


● Extractor de Conceptos

Este módulo permite realizar la identificación de los posibles conceptos de un determinado trecho de un objeto el texto, la misma estrategia de la obtención de un tema central.


● Minerador Completo

Este es el sistema completo que realiza la integración de todos los módulos anteriores, haciendo posible identificación del tema central de un texto, la identificación de los conceptos presentes en el mismo y la exhibición en forma de Los gráficos.


● Visualizar datos procesados ​​

A través de este módulo es posible buscar datos que ya se han procesado y se almacenan en la base de datos datos de la aplicación.
